# Flask Blog Platform

A simple full-stack blogging platform built using Flask and SQLite.

## Features

- Create Blog Posts
- Read Blog Posts
- Update Blog Posts
- Delete Blog Posts
- Responsive UI
- SQLite Database
- Form Validation

## Tech Stack

### Frontend
- HTML
- CSS
- Jinja2

### Backend
- Python Flask
- SQLite
- SQLAlchemy
